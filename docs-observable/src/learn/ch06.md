# The BJT

```js
import {kdlToSvg, SimBuilder} from "../components/spice.js";
import {Resistance, Voltage, formatSpice, formatEE} from "../components/ee-inputs.js";
import {simPanel} from "../components/readout.js";
```

The bipolar junction transistor was the workhorse of electronics for three decades before CMOS took over digital circuits. It remains indispensable in analog design: operational amplifiers, voltage references, bandgap circuits, high-speed ECL logic, and RF front-ends all rely on BJTs.

Where the MOSFET is a voltage-controlled switch, the BJT is a *current-controlled current amplifier*. A small current into the base produces a much larger current between collector and emitter, with a gain (${tex`\beta`}) that is remarkably well-controlled by the fabrication process.

## Two flavors

**NPN:** Current flows from collector to emitter, controlled by base current. The emitter arrow points *out* of the device. This is the more common type.

**PNP:** Current flows from emitter to collector, controlled by base current. The emitter arrow points *in*. Complementary to NPN, like PMOS is to NMOS.

The BJT has three terminals: base (B), collector (C), and emitter (E). Unlike the MOSFET, the control terminal (base) draws significant current -- this is the fundamental difference between field-effect and bipolar devices.

### The exponential law

The BJT's collector current follows an exponential relationship with base-emitter voltage:

```tex
I_C = I_S \cdot \exp\left(\frac{V_{BE}}{V_T}\right)
```

where ${tex`I_S`} is the saturation current (typically ${tex`10^{-15}`} to ${tex`10^{-12}`} A) and ${tex`V_T = kT/q \approx 26`} mV at room temperature.

This exponential relationship is extraordinarily precise -- it holds over many decades of current. It gives the BJT its transconductance:

```tex
g_m = \frac{I_C}{V_T}
```

At ${tex`I_C = 1`} mA, ${tex`g_m = 38.5`} mS. Compare this to a MOSFET, where achieving the same ${tex`g_m`} requires much more current or a very wide device. This intrinsic transconductance advantage is why BJTs dominate in precision analog circuits.

### The four operating regions

| Region | BE Junction | BC Junction | Behavior |
|--------|-----------|-----------|----------|
| Forward active | Forward biased | Reverse biased | Normal amplification |
| Reverse active | Reverse biased | Forward biased | Poor amplification (rarely used) |
| Saturation | Forward biased | Forward biased | Both junctions on, ${tex`V_{CE} \approx 0.2`} V |
| Cutoff | Reverse biased | Reverse biased | Off |

Forward active is the amplification region -- where the collector current is ${tex`\beta`} times the base current. Saturation (not the same meaning as MOSFET saturation) is where the BJT is used as a switch in the "on" state.

## The Gummel-Poon Model

The Gummel-Poon model (1970) is to the BJT what BSIM3 is to the MOSFET: the standard model that every SPICE simulator implements.

### The transport current

At the heart of the model is the *transport current*:

```tex
I_T = I_S \left( \exp\left(\frac{V_{BE}}{N_F \cdot V_T}\right) - \exp\left(\frac{V_{BC}}{N_R \cdot V_T}\right) \right)
```

where:

| Parameter | Symbol | Typical NPN | Meaning |
|-----------|--------|-------------|---------|
| IS | ${tex`I_S`} | ${tex`10^{-15}`} A | Saturation current |
| NF | ${tex`N_F`} | 1.0 | Forward emission coefficient |
| NR | ${tex`N_R`} | 1.0 | Reverse emission coefficient |

This is the Ebers-Moll transport equation. In forward active operation (${tex`V_{BE} > 0`}, ${tex`V_{BC} < 0`}), the second exponential is negligible and ${tex`I_T \approx I_S \exp(V_{BE}/N_F V_T)`}.

### Forward and reverse beta

The base current is the sum of forward and reverse components:

```tex
I_B = \frac{I_S}{B_F} \left( \exp\left(\frac{V_{BE}}{N_F \cdot V_T}\right) - 1 \right) + \frac{I_S}{B_R} \left( \exp\left(\frac{V_{BC}}{N_R \cdot V_T}\right) - 1 \right)
```

where ${tex`B_F`} is the ideal maximum forward beta (typically 100) and ${tex`B_R`} is the reverse beta (typically 1).

### Base charge modulation

The Gummel-Poon model's key innovation is the *normalized base charge* ${tex`q_b`}, which modulates the transport current:

```tex
I_T = \frac{I_S}{q_b} \left( \exp\left(\frac{V_{BE}}{N_F \cdot V_T}\right) - \exp\left(\frac{V_{BC}}{N_R \cdot V_T}\right) \right)
```

The base charge factor is:

```tex
q_b = \frac{q_1}{2} \left( 1 + \sqrt{1 + 4 q_2} \right)
```

where:

```tex
q_1 = 1 + \frac{V_{BE}}{V_{AF}} + \frac{V_{BC}}{V_{AR}}
```

```tex
q_2 = \frac{I_S}{IKF} \exp\left(\frac{V_{BE}}{N_F V_T}\right) + \frac{I_S}{IKR} \exp\left(\frac{V_{BC}}{N_R V_T}\right)
```

The ${tex`q_1`} term captures the **Early effect**: as ${tex`V_{CE}`} increases, the base width narrows, ${tex`q_1`} decreases, and ${tex`I_T`} increases. The ${tex`q_2`} term captures **high-injection effects**: at large currents, gain rolls off.

### Gummel plot

The Gummel plot shows ${tex`I_C`} and ${tex`I_B`} on a log scale vs. ${tex`V_{BE}`}. It reveals the three regimes of beta: low-current recombination, the ideal region, and high-injection rolloff.

```js
const bf = view(Inputs.range([50, 500], {value: 100, step: 10, label: "BF (forward beta)"}));
const isParam = view(Inputs.range([-16, -12], {value: -14, step: 0.5, label: "log10(IS)"}));
const vaf = view(Inputs.range([20, 200], {value: 100, step: 10, label: "VAF (V)"}));
```

```js
{
  const IS = Math.pow(10, isParam);
  const VT = 0.026;
  const gummelData = [];
  for (let vbe = 0.3; vbe <= 0.8; vbe += 0.005) {
    const ic = IS * Math.exp(vbe / VT);
    const ib = ic / bf;
    if (ic > 1e-15 && ib > 1e-15) {
      gummelData.push({vbe, current: ic, label: "IC"});
      gummelData.push({vbe, current: ib, label: "IB"});
    }
  }
  display(Plot.plot({
    width: 560,
    height: 400,
    color: {legend: true},
    x: {label: "VBE (V)", domain: [0.3, 0.8]},
    y: {label: "Current (A)", type: "log", grid: true},
    marks: [
      Plot.line(gummelData, {x: "vbe", y: "current", stroke: "label", strokeWidth: 2})
    ]
  }));
}
```

The parallel slopes on the log scale (both lines are straight with the same slope) confirm the exponential relationship. The vertical separation is the current gain ${tex`\beta = I_C / I_B`}.

### How the BJT stamps into MNA

The linearized BJT becomes a companion model for Newton-Raphson with key conductances:

- ${tex`g_m = \partial I_C / \partial V_{BE}`} -- transconductance
- ${tex`g_o = \partial I_C / \partial V_{CE}`} -- output conductance (Early effect)
- ${tex`g_\pi = \partial I_B / \partial V_{BE}`} -- input conductance
- ${tex`g_\mu = \partial I_B / \partial V_{BC}`} -- reverse feedback

These stamp into the MNA matrix at the (collector, emitter), (base, emitter), and (base, collector) positions, along with equivalent current sources.

The exponential nonlinearity of the BJT makes Newton-Raphson convergence more challenging than for MOSFETs. A 26 mV change in ${tex`V_{BE}`} changes ${tex`I_C`} by a factor of ${tex`e \approx 2.718`}. The simulator must use voltage limiting and careful initial-guess strategies to converge reliably.

## The Early Effect

In the ideal BJT, collector current depends only on ${tex`V_{BE}`}. But real BJTs show a slight increase of ${tex`I_C`} with ${tex`V_{CE}`}. This is the *Early effect*.

### The physical picture

The collector-base junction has a depletion region. When ${tex`V_{CE}`} increases, this depletion region widens, eating into the neutral base. A shorter effective base width means more carriers reach the collector, so ${tex`I_C`} increases slightly.

### The Early voltage

If you extrapolate the ${tex`I_C`} vs. ${tex`V_{CE}`} curves backward, they converge at a single point on the negative ${tex`V_{CE}`} axis. This intercept is the *forward Early voltage* ${tex`V_{AF}`}:

```tex
I_C = I_S \exp\left(\frac{V_{BE}}{V_T}\right) \left(1 + \frac{V_{CE}}{V_{AF}}\right)
```

### Output resistance

The Early effect gives the BJT a finite output resistance:

```tex
r_o = \frac{V_{AF} + V_{CE}}{I_C} \approx \frac{V_{AF}}{I_C}
```

For a BJT biased at ${tex`I_C = 1`} mA with ${tex`V_{AF} = 100`} V:

```tex
g_m = \frac{1 \text{ mA}}{26 \text{ mV}} = 38.5 \text{ mS}, \qquad r_o = \frac{100 \text{ V}}{1 \text{ mA}} = 100 \text{ k}\Omega
```

The intrinsic gain is ${tex`g_m \cdot r_o = 3{,}850`} -- nearly 4000. This is one reason BJTs excel in precision analog circuits. A MOSFET at the same current achieves an intrinsic gain of perhaps 20-50.

## Parasitics

A real BJT has parasitic elements that affect its high-frequency behavior: series resistances, junction capacitances, and transit times.

### Series resistances

| Parameter | Symbol | Typical NPN | Meaning |
|-----------|--------|-------------|---------|
| RB | ${tex`R_B`} | 10-100 ${tex`\Omega`} | Base resistance |
| RE | ${tex`R_E`} | 1-5 ${tex`\Omega`} | Emitter resistance |
| RC | ${tex`R_C`} | 10-50 ${tex`\Omega`} | Collector resistance |

**Base resistance (RB)** is the most important parasitic. It creates a voltage drop between the external base and the intrinsic junction, causing effective ${tex`\beta`} to decrease at high currents.


### Junction capacitances

Each PN junction has a depletion capacitance:

```tex
C_J(V) = \frac{C_{J0}}{(1 - V/V_J)^{M_J}}
```

The base-collector capacitance ${tex`C_{JC}`} creates the *Miller effect*: the effective input capacitance is multiplied by the voltage gain:

```tex
C_{in,Miller} = C_{JC} \cdot (1 + |A_v|)
```

This Miller multiplication is the dominant bandwidth limitation in common-emitter amplifiers.

### Transit times

When carriers traverse the base, a *diffusion capacitance* arises:

```tex
C_{diff} = \tau_F \cdot g_m
```

where ${tex`\tau_F`} is the forward transit time (parameter TF). This determines the transition frequency ${tex`f_T`}:

```tex
f_T = \frac{1}{2\pi \left(\tau_F + \frac{C_{JE} + C_{JC}}{g_m}\right)}
```

Typical ${tex`f_T`} values range from 1 GHz (standard process) to 300+ GHz (SiGe HBTs).

## Amplifier Circuits

The BJT's high transconductance and well-controlled current gain make it the natural choice for analog amplifier design. The common-emitter configuration is the most widely used topology.

### The common-emitter amplifier

```js
const ceVcc = view(Voltage({label: "VCC", value: 12, max: 20}));
const ceVin = view(Voltage({label: "Vin", value: 0.7, min: 0.5, max: 0.8, step: 0.01}));
const ceRc = view(Resistance({label: "RC", value: 1000}));
```

```js
const ceKdl = `circuit "Common-Emitter Amplifier" {
    group "amplifier" {
        component "RC" type="resistor" { value "${formatSpice(ceRc)}"; port "1" net="vcc"; port "2" net="vout"; place col=0 row=0 }
        component "Q1" type="npn" {
            model "NPN"
            port "collector" net="vout"
            port "base" net="vin"
            port "emitter" net="gnd"
            place col=0 row=2
        }
        component "Vin" type="voltage-source" { value "DC ${ceVin}"; port "pos" net="vin"; port "neg" net="gnd"; place col=-3 row=3 }
    }
    node "vcc" role="supply" voltage="${ceVcc}" label="VCC"
    node "gnd" role="ground"
    node "vout" label="Vout"
}`;
const ceSim = await SimBuilder.fromKdl(ceKdl)
  .op()
  .measure("Vout", "voltage", "vout")
  .withSvg()
  .run();
display(simPanel(ceSim));
```

This is the simplest form: a single NPN transistor with a collector resistor. The input voltage ${tex`V_{in}`} drives the base; the output is taken at the collector.

### DC operating point

With ${tex`V_{BE} = V_{in}`} = ${ceVin} V, SPICE computes:

```js
{
  const IS = 1e-15;
  const VT = 0.026;
  const ic = IS * Math.exp(ceVin / VT);
  const vout_calc = ceVcc - ic * ceRc;
  const gm = ic / VT;
  const av = -gm * ceRc;
  display(html`<p>${tex`I_C = I_S \exp(V_{BE}/V_T)`} = ${formatEE(ic, "A")}<br>
${tex`V_{out} = V_{CC} - I_C R_C`} = ${vout_calc.toFixed(2)} V<br>
${tex`g_m = I_C / V_T`} = ${formatEE(gm, "S")}<br>
${tex`A_v = -g_m R_C`} = ${av.toFixed(1)}</p>`);
}
```

The negative gain means the output is inverted relative to the input. A small increase in ${tex`V_{BE}`} causes a large increase in ${tex`I_C`}, which pulls ${tex`V_{out}`} down.

### The three BJT amplifier topologies

| Topology | Voltage gain | Input impedance | Output impedance | Use |
|----------|-------------|-----------------|------------------|-----|
| Common-emitter | ${tex`-g_m R_C`} (high) | ${tex`r_\pi`} (medium) | ${tex`R_C`} (medium) | General amplification |
| Common-base | ${tex`g_m R_C`} (high) | ${tex`1/g_m`} (low) | ${tex`R_C`} (medium) | High-frequency, cascode |
| Common-collector | ${tex`\approx 1`} (unity) | ${tex`\beta r_o`} (very high) | ${tex`1/g_m`} (low) | Impedance buffer |

Each topology has the same BJT, the same Gummel-Poon model, the same MNA stamps. What changes is the circuit around it -- which terminal is the input, which is the output, which is the AC ground. SPICE does not know or care about the topology; it simply solves the matrix.

### Biased common-emitter amplifier

The simple circuit above has a serious problem: the DC operating point is extremely sensitive to ${tex`V_{BE}`}. A 10 mV change changes ${tex`I_C`} by about 50%. Practical amplifiers use feedback biasing with a voltage divider and emitter resistor:

```js
const biasVcc = view(Voltage({label: "VCC", value: 12, max: 20}));
const biasRb1 = view(Resistance({label: "RB1", value: 10000}));
const biasRb2 = view(Resistance({label: "RB2", value: 2200}));
const biasRc = view(Resistance({label: "RC", value: 1000}));
const biasRe = view(Resistance({label: "RE", value: 220}));
```

```js
const biasKdl = `circuit "Biased CE Amplifier" {
    group "amplifier" {
        component "RB1" type="resistor" { value "${formatSpice(biasRb1)}"; port "1" net="vcc"; port "2" net="base"; place col=0 row=0 }
        component "RB2" type="resistor" { value "${formatSpice(biasRb2)}"; port "1" net="base"; port "2" net="gnd"; place col=0 row=2 }
        component "RC" type="resistor" { value "${formatSpice(biasRc)}"; port "1" net="vcc"; port "2" net="vout"; place col=3 row=0 }
        component "Q1" type="npn" {
            port "collector" net="vout"
            port "base" net="base"
            port "emitter" net="emitter"
            place col=3 row=2
        }
        component "RE" type="resistor" { value "${formatSpice(biasRe)}"; port "1" net="emitter"; port "2" net="gnd"; place col=3 row=4 }
        wire "base" {
            waypoint col=0 row=2
            waypoint col=2 row=2
            waypoint col=2 row=3
        }
    }
    node "vcc" role="supply" voltage="${biasVcc}" label="VCC"
    node "gnd" role="ground"
    node "vout" label="Vout"
}`;
const biasSim = await SimBuilder.fromKdl(biasKdl)
  .op()
  .measure("VCC", "voltage", "vcc")
  .measure("Vout", "voltage", "vout")
  .measure("Vbase", "voltage", "base")
  .measure("IC", "current", "RC")
  .withSvg()
  .run();
display(simPanel(biasSim));
```

The voltage divider (R1, R2) sets the base voltage. The emitter resistor RE provides negative feedback: if ${tex`I_C`} increases, the voltage across RE increases, reducing ${tex`V_{BE}`}, which reduces ${tex`I_C`} back down. The price is reduced gain:

```tex
A_v = -g_m \cdot \frac{R_C}{1 + g_m R_E} \approx -\frac{R_C}{R_E}
```

For AC amplification, RE is often bypassed with a capacitor to restore the full gain at signal frequencies while keeping the DC stabilization.

## In spice-rs

The BJT device model lives in `device/bjt.rs`. It implements the Gummel-Poon model including transport current with forward and reverse components, base charge modulation (Early effect + high-injection), junction capacitances (depletion + diffusion), series resistances (RB, RE, RC), and transit times (TF, TR).

The BJT model is simpler than BSIM3/4 in parameter count but has its own subtleties, particularly in the treatment of base charge and the smooth transitions between operating regions.
